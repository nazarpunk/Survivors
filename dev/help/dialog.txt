/*
Диалоги они же варианты выбора задаются модулем dialog.
Основываются на переходах label
*/

//Пометим начало
label start

//Ключевое слово что-бы начать набор диалога
dialog

//Здесь указываем ярлыки перехода

dialog_01_kitchen Кухня
dialog_01_control Комната управления
dialog_01_bedroom Спальня

//Ключевое слово что-бы закончить набор диалога
dialog

//Ярлык куда ведёт выбор варианта диалога
label dialog_01_kitchen
bg sun/kitchen
Вы пришли на кухню
Пора двигаться дальше
go dialog_01_end

//Ярлык куда ведёт выбор варианта диалога
label dialog_01_control
bg sun/control
Вы пришли в комнату управления
Пора двигаться дальше
go dialog_01_end

//Ярлык куда ведёт выбор варианта диалога
label dialog_01_bedroom
bg sun/bedroom_chaos
Вы пришли в спальню
Пора двигаться дальше
go dialog_01_end

//Продолжаем историю отсюда
label dialog_01_end
bg sun/meadow
Мы вышли из корабля
Зайдём обратно

//Возвращаемся в начало
go start
